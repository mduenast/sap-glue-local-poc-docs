# Limitations

This repository and PoC are intentionally limited.

## Explicitly out of scope

- No real SAP connectivity.
- No real SNP Glue execution.
- No real AWS infrastructure.
- No real Snowflake external stage.
- No production guarantees for data volumes, performance, resilience, observability, governance, or security.
- No validation of any vendor-specific runtime behavior.
- No functional design validation for any target dbt project.
- No client-specific, employer-specific, internal, proposal, or production system content.

The value of the PoC is in clarifying contracts and responsibilities. It should not be used as proof that a production implementation is ready.
