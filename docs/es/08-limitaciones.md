# Limitaciones

Este repositorio y la PoC estan limitados de forma intencionada.

## Fuera de alcance explicito

- No hay conectividad SAP real.
- No hay ejecucion real de SNP Glue.
- No hay infraestructura AWS real.
- No hay external stage real de Snowflake.
- No hay garantias productivas sobre volumenes, rendimiento, resiliencia, observabilidad, gobierno o seguridad.
- No hay validacion de comportamiento de runtimes especificos de proveedor.
- No hay validacion del diseno funcional de ningun proyecto dbt destino.
- No hay contenido especifico de cliente, empleador, proyecto interno, propuesta o sistema productivo.

El valor de la PoC esta en aclarar contratos y responsabilidades. No debe usarse como prueba de que una implementacion productiva esta lista.
