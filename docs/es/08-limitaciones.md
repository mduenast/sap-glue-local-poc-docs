# Limitaciones

Este repositorio y la PoC estan limitados de forma intencionada.

## Fuera de alcance explicito

- Conectividad SAP real.
- Ejecucion real de SNP Glue.
- Infraestructura AWS real.
- External stages reales de Snowflake.
- Diseno funcional de cualquier proyecto dbt destino.
- Volumenes productivos, rendimiento, resiliencia, observabilidad, gobierno y seguridad.
- Cualquier contenido especifico de cliente, empleador, proyecto interno, propuesta o sistema productivo.

El valor de la PoC esta en aclarar contratos y responsabilidades. No debe usarse como prueba de que una implementacion productiva esta lista.
